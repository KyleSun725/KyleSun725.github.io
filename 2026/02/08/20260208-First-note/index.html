关于随笔

为什么要写随笔

  • 实用主义视角下，我的记忆会随着年龄增长而衰弱，我的思维也许会随着履历而不断演化，但我坚持相信，当下的我的想法应当被记录，不论是作为反思的审视对象，还是作为日后的准绳，都值得被记住。

  • 浪漫主义视角下，如果很久以后，能一窥以前的自己是怎么想的，也是一件值得高兴的事。再者，让自己的想法在网络空间占有一席之地，也是一件不错的事情。

记录思想

  • 杂记

    • 不知从何开始，先想到什么写什么吧

    • 任何表述均没有含沙射影，写了什么就是且仅是什么。

    • 成年以前的人生：从2000~2018，一路刺头+一路优秀。以前很瘦，也许有点小帅。刺头惹人厌、优秀讨人喜，一路以来我的人际关系好的很好，坏的断联，以前我会觉得这般很好，到目前为止似乎发生了一点细微的变化，我慢慢也会去关注断联的那部分。

    • 本科教会我的：2018，运气使然上了很好的大学。尽管我从中学到了很多、收获了很多，但也会想，要是给我重来一次的机会，我会让那四年过得更加精彩。

      • 广交益友：朋友是很珍贵的资源。（虽然这个说法有点功利）
      • 认清自己：人外有人天外有天（此处应有参考文献），但切忌妄自菲薄。
      • 问清自己：想好自己要什么，或至少得到余集。不过这个很难，需要大量的外出接触，其实现在都快26岁了，我也只是模模糊糊地摸到了自己想要什么，清清楚楚地知道了自己不想要什么。
      • 坚持自己：是非对错自己判断。再好的学校也有不配得者，再差的学校也有脱颖而出之人，尽管统计规律客观存在，但也要承认特殊情况。

    • 读研教会我的：2022，继续上了很好的大学，但是调剂的过程中没有遵循上述的‘坚持自己’，导致被动换了个不是最喜欢的专业（资源与环境，至少在2025年还是比较难找工作的，而且我的兴趣也在能源动力那边）。不过，也无可厚非，毕竟也没有人可以判断我在那一个branch中能实现什么。

      • 潜心做事（坚持自己pro max）：不管其它人在干什么，做好自己该做的事情。
      • 发掘自己：抓住自己擅长的事情，构建自己的护城河，让自己具备独有的稀缺性。
      • 照顾好身边人：我做到了一个正常人该做的，但是称不上是一个好的伴侣，而后者才是我希望的。

    • 综上所述，目前的特质：

      1. 坚持做（自以为）正确的事、说（自以为）正确的话
      2. 在少数关系很近的人面前，第一条作废
      3. ESTJ，其中E岌岌可危
      4. 胃口不好，吃软不吃硬
      5. 有理想，赚钱是理想之一
      6. 慕强，唯技术论

捕捉灵感

  • 当前开发的BMS2Brick能够实现在不背板的前提下完美复刻Ground Truth，其中第一轮抓entity，第二轮抓topology。每一轮内部都会经历多个epoch，给出错误与正确的答案并记录思考过程。

  • Q:这么折腾，为什么不直接背板呢？

    • A:Ground Truth不是什么时候都有的。假如Ground Truth已经存在，也就没有开发这个pipeline的意义了。

  • Q:当前这个pipeline的意义是什么呢？

    • A:提供了一种思路，即利用Ground Truth积累较多的迭代经验，从而在没有Ground Truth的情况下尝试提升brick model的建模准确性。
